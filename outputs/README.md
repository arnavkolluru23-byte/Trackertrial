# UW Life + Informatics Tracker

This folder is ready to be deployed to GitHub Pages.

## Publish steps

1. Create a GitHub repository.
2. Push this project to the `main` branch.
3. In GitHub, open `Settings` -> `Pages`.
4. Set `Source` to `GitHub Actions`.
5. Push again if needed, then wait for the `Deploy GitHub Pages` workflow to finish.

The published site will serve this folder as the live Pages site.
